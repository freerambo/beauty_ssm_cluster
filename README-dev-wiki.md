##JAXB 库用法（用于xml格式化）
- 注意：主要使用
    - `XmlRootElement`定义根节点;
    - `XmlElement`定义其他节点，要注意定义在get方法上。
    
- [官方文档](https://jaxb.java.net/tutorial/index.html)
- [反序列化api，含示例](https://docs.oracle.com/javase/7/docs/api/javax/xml/bind/Unmarshaller.html)
